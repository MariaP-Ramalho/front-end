# 📊 Live Insights App 

Bem-vindo ao **Live Insights App**! 😎

## 🚀 Primeiros Passos 

1. **Instale as dependências** 

   ```bash
   npm install
   ```

2. **Dê a partida no projeto** 

    ```bash
    npm start
    ```

3. **Agora, é só abrir seu navegador e acessar:**

    http://localhost:3000

A página vai recarregar automaticamente quando fizer mudanças.
